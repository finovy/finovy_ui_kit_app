# Finovy UIKit 示例工程

轻量、可靠的移动端Flutter组件库

[English](README.md) | 中文版
## **一、使用方法**
在pubspec.yaml文件dependencies节点加入：
```dart
fn_ui_kit:
  git:
    url: https://github.com/finovy/finovy_ui_kit.git
```
